📘 Как работают операции с матрицами
➕ Сложение (A+B)

Складываем соответствующие элементы. Требуются матрицы одинакового размера.

➖ Вычитание (A−B)

Вычитаем соответствующие элементы. Требуются матрицы одинакового размера.

✖️ Умножение (A×B)

Строка A на столбец B. Число столбцов A = числу строк B.

🔢 Умножение на число (k·A)

Каждый элемент матрицы умножается на число.

🔄 Транспонирование (Aᵀ)

Строки становятся столбцами.

📊 Определитель (det A)

Числовая характеристика квадратной матрицы.

⚡ Обратная матрица (A⁻¹)

A·A⁻¹ = E. Только для квадратных матриц с det≠0.

📈 Ранг матрицы

Максимальное число линейно независимых строк.

📏 След матрицы

Сумма элементов на главной диагонали.

🔋 Возведение в степень (Aⁿ)

Последовательное умножение на себя. Только для квадратных.
